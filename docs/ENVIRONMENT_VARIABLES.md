# Environment Variables

This project reads environment variables from `.env` through Laravel config and a small number of controller-level overrides.

Use `.env.example` as the starting point, then add integration-specific values from this document.

## Minimal Local Setup

These keys are enough to boot the app locally and use JWT-protected APIs.

| Variable | Example / Default | Required | Purpose |
| --- | --- | --- | --- |
| `APP_NAME` | `Airventory` | Yes | Application name used in logs/email/session prefixes. |
| `APP_ENV` | `local` | Yes | Runtime environment. |
| `APP_KEY` | generated by `php artisan key:generate` | Yes | Laravel encryption key. |
| `APP_DEBUG` | `true` (local) | Yes | Error verbosity. |
| `APP_URL` | `http://localhost` | Yes | Base URL used in links and storage URLs. |
| `DB_CONNECTION` | `sqlite` or `mysql` | Yes | Database driver. |
| `DB_DATABASE` | `database/database.sqlite` or db name | Yes | Database path/name. |
| `DB_HOST` | `127.0.0.1` | MySQL/Postgres/SQL Server | DB host. |
| `DB_PORT` | `3306` (mysql) | MySQL/Postgres/SQL Server | DB port. |
| `DB_USERNAME` | `root` | MySQL/Postgres/SQL Server | DB username. |
| `DB_PASSWORD` | empty/local secret | MySQL/Postgres/SQL Server | DB password. |
| `JWT_SECRET` | generated by `php artisan jwt:secret` | Yes for JWT APIs | JWT token signing secret. |
| `QUEUE_CONNECTION` | `database` or `redis` | Recommended | Queue backend. |
| `CACHE_STORE` | `database` | Recommended | Cache backend. |
| `SESSION_DRIVER` | `database` | Recommended | Session storage. |

## Authentication and JWT

| Variable | Default | Notes |
| --- | --- | --- |
| `JWT_TTL` | `60` | Default access token TTL (minutes). |
| `JWT_REMEMBER_TTL` | `43200` | "Remember me" token TTL (minutes). |
| `JWT_REFRESH_TTL` | `20160` | Refresh window (minutes). |
| `JWT_ALGO` | `HS256` | Signing algorithm. |
| `JWT_BLACKLIST_ENABLED` | `true` | Enables token invalidation on logout. |
| `JWT_BLACKLIST_GRACE_PERIOD` | `0` | Grace period for concurrent requests. |
| `JWT_LEEWAY` | `0` | Clock skew tolerance. |
| `ADMIN_JWT_TTL` | falls back to `JWT_TTL` | Admin token lifetime override. |
| `ADMIN_JWT_REFRESH_TTL` | falls back to `JWT_REFRESH_TTL` | Admin refresh lifetime override. |
| `JWT_PUBLIC_KEY` / `JWT_PRIVATE_KEY` / `JWT_PASSPHRASE` | unset | Used for asymmetric JWT setups. |

### Order Sync Scheduler

| Variable | Default | Purpose |
| --- | --- | --- |
| `ORDER_SYNC_ENABLED` | `true` | Enables scheduled missing-order sync. |
| `ORDER_SYNC_SCHEDULE` | `0 2 * * *` | Cron for `orders:sync-missing`. |
| `ORDER_SYNC_QUEUE` | `low` | Queue used by order sync jobs. |
| `ORDER_SYNC_DEFAULT_DAYS` | `7` | Default lookback days. |
| `ORDER_SYNC_MAX_DAYS` | `90` | Upper bound for manual `--days`. |
| `ORDER_SYNC_MIN_HOURS` | `6` | Minimum interval between syncs for same store. |
| `ORDER_SYNC_LOOKBACK_MINUTES` | `10` | Safety lookback window for edge times. |
| `ORDER_SYNC_LOCK_SECONDS` | `1800` | Command overlap lock window. |
| `ORDER_SYNC_CHUNK_SIZE` | `50` | Batch size for processing. |
| `ORDER_SYNC_CHUNK_DELAY_MS` | `100` | Delay between chunks. |

| Variable | Purpose |
| --- | --- |
| `FILESYSTEM_DISK` | Default storage disk (`local`, `s3`, etc.). |
| `AWS_ACCESS_KEY_ID` / `AWS_SECRET_ACCESS_KEY` | AWS credentials for S3/SQS/SES/cache drivers. |
| `AWS_DEFAULT_REGION` | AWS region. |
| `AWS_BUCKET` | S3 bucket name. |
| `AWS_URL` / `AWS_ENDPOINT` | Custom S3 endpoint/CDN URL. |

## Notes

- Keep secrets in `.env` only; do not commit them.
- Variables not set will use Laravel/config defaults where provided.
- If you change env values in production, refresh config cache:

```bash
php artisan config:clear
php artisan config:cache
```
